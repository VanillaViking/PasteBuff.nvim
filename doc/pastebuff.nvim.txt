pastebuff.setup({address}, {port})                             *pastebuff.setup*
    Setup connection with a PasteBuff server

    Parameters: ~
        {address}  (string)
        {port}     (number)


pastebuff.size()                                                *pastebuff.size*
    Get the number of buffers in the connected PasteBuff server

    Returns: ~
        (number)  size


pastebuff.get({buf_name})                                        *pastebuff.get*
    Get the contents of the specified buffer name

    Parameters: ~
        {buf_name}  (string)


pastebuff.set({buf_name}, {value})                               *pastebuff.set*
    Set the contents of the specified buffer with value

    Parameters: ~
        {buf_name}  (string)
        {value}     (string)
